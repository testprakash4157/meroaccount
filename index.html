⚠️ First-time setup: You need to deploy the Google Apps Script backend and enter the URL below.
Steps:
1. Open Google Sheets → New spreadsheet
2. Extensions → Apps Script
3. Paste the code from gas_backend.js
4. Deploy → New Deployment → Web App
5. Execute as: Me · Access: Anyone
6. Copy the Web App URL and paste below

The URL is saved in your browser's localStorage
